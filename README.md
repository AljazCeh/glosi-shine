# Avtopralnica Glosi - spletna predstavitev

Ta repozitorij vsebuje premium, minimalistično stran za Avtopralnico Glosi (ročna avtopralnica v Ivančni Gorici) z optimizacijo za SEO, local business schema in dostopnost. Stran je zgrajena v React + Vite + TypeScript in je pripravljena za produkcijsko objavo. 

## Zaženite lokalno

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
